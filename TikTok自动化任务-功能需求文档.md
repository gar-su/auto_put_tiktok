# TikTok 自动化任务 — 功能需求文档
## 1. 页面结构
### 1.1 顶部导航
顶部栏固定显示「TikTok自动化投放」Logo 与面包屑导航。
外层 Tab：
- 自动化任务 — 默认选中，下方显示内层 Tab
- 前置配置 — 切换到前置配置页（账户包管理、定向包管理）
内层 Tab（仅自动化任务下显示）：
- 任务 — 自动化任务列表，默认选中
- 广告系列 — 按日/按月筛选的广告系列列表
- 广告组 — 广告组列表
- 广告 — 广告列表
### 1.2 自动化任务列表页
工具栏：任务ID、任务名称、任务状态、竞价策略、创建人、最高数量、单次成效费用目标、竞价上限、广告花费回报目标 筛选字段 + 查询/重置按钮 +「+ 新建任务」按钮。
底部批量操作栏：已选择 N 个任务 + 批量启动/批量暂停。
表格列：复选框、ID、任务名称、状态、竞价策略、日预算、执行间隔(分钟)、优化目标、成效目标、渠道号、上次完成时间、创建人、更新人、创建时间、更新时间、操作。
### 1.3 广告系列列表页
工具栏：按日/按月切换、日期范围、任务ID、任务名称、上传广告系列ID、账户名称、系列名称、广告系列状态、短剧、渠道号、竞价策略、业务类型 筛选字段 + 查询/重置。
表格列：复选框、ID、系列名称、系列ID、状态、投放账户ID/名称、单日预算、竞价策略、版位、优化目标、创建时间、操作。
### 1.4 广告组列表页
表格列：复选框、ID、广告组名称、广告组ID、状态、单日预算、竞价策略、版位、优化目标、创建时间、操作。
### 1.5 广告列表页
表格列：复选框、ID、广告名称、广告ID、状态、关联商品库、行动号召、创建时间、操作。
### 1.6 前置配置页
两个子页：
- 账户包管理 — 搜索栏 + 表格（复选框、ID、账户包名称、业务类型、渠道名称、渠道号、投放策略、回传模板、像素名称、创建时间、更新时间、操作）+「+ 新增账户包」按钮
- 定向包管理 —「+ 新建定向包」按钮 + 列表
## 2. 新建任务 — 基本信息
点击「+ 新建任务」弹出模态框，包含以下表单区域：
### 2.1 任务名称
文本输入，必填，最大 50 字符，带字数统计 `0 / 50`。
### 2.2 状态
按钮组 `btn-group`，二选一：
| 按钮文案 | data-value | 默认 |
|---------|-----------|------|
| 启用 | `1` | |
| 暂停 | `0` | ✓ |
### 2.3 时区与投放时段
时区下拉选择（硬编码选项：UTC-8, UTC+0, UTC+8 等）。
投放时段：可动态添加/删除的时间范围行，每行含开始时间、结束时间两个 `time` 输入框。「+ 添加时段」按钮追加新行，每行右侧有删除按钮。至少保留一个时段行。
### 2.4 执行频率
数字输入，单位分钟，必填。占位提示文案「请输入执行频率」。
### 2.5 结束运行条件
两个条件行，每行格式：`累计投放 [指标下拉] ≥ [数值输入]`
- 条件1：指标固定为「短剧数」，数值必填
- 条件2：指标固定为「账户数」，数值必填
### 2.6 单轮单短剧计划创建上限
数字输入，范围 1-30，不填默认 30。占位提示「不填写默认30」。
### 2.7 短剧选择
搜索区域含三个输入项：
- Book ID 文本框
- 短剧名称 文本框
- 语言 下拉（不限、中文、英文、泰文、马来文 等）
点击「搜索」按钮后展示匹配结果卡片，卡片显示：
- 短剧信息：剧本ID、剧本名、语言、类型、标签、简介（双列网格布局）
- 每张卡片下方「添加」按钮，已添加的按钮 disabled
已添加的短剧在下方「已选短剧」区域以标签展示，可逐个移除（× 按钮）。
## 3. 新建任务 — 素材数据筛选
### 3.1 筛选模式
按钮组，二选一：
| 按钮文案 | data-value | 默认 | 行为 |
|---------|-----------|------|------|
| 全部 | `all` | ✓ | 隐藏 3.2 所有自定义字段 |
| 自定义 | `custom` | | 显示 3.2 所有自定义字段 |
### 3.2 自定义筛选字段
仅在筛选模式 = 自定义时可见：
- **业务类型** 按钮组：全部(`all`, 默认) / 免费(`free`) / 付费(`paid`)
- **筛选媒体** 按钮组：全部(`all`, 默认) / TikTok(`tiktok`) / Pangle(`pangle`)
- **素材创建时间** 按钮组：最近7天(`7`, 默认) / 最近15天(`15`) / 最近30天(`30`)
- **统计数据范围** 按钮组：最近7天(`7`, 默认) / 最近15天(`15`) / 最近30天(`30`)
- **素材数量** 按钮组：不限(`unlimit`, 默认) / 自定义(`custom`) — 选自定义时显示数量输入框
- **筛选条件**：动态条件行列表，每行含：
  - 指标下拉：消耗(`cost`) / DOROI(`doroi`) / D0广告ROI(`d0_ad_roi`) / D0 IAA(`d0_iaa`) / 综合达标率(`compliance`) / 老达标率(`old_compliance`)
  - 运算符下拉：`>=`、`>`、`=`、`<`、`<=`、`介于`
  - 值输入框（`介于` 时显示两个输入框，其他显示一个）；DOROI/综合达标率/老达标率/D0广告ROI 带 `%` 后缀
  - 删除按钮（×）
  - 约束：同一指标不可重复添加，重复时显示红色错误提示「指标重复，请修改」
  - 「+ 添加条件」按钮追加新行
  - 筛选模式=自定义且无条件行时，显示红色错误提示「选择自定义时，至少需要输入一个条件」
- **按指标排序** 按钮组：消耗(`cost`, 默认) / DOROI(`doroi`) / D0广告ROI(`d0_ad_roi`) / D0 IAA(`d0_iaa`) / 综合达标率(`compliance`)

### 3.3 单剧本/单账户最高重复次数
按钮组：
| 按钮文案 | data-value | 默认 |
|---------|-----------|------|
| 不限 | `unlimit` | ✓ |
| 自定义 | `custom` | |

选自定义时弹出数字输入框。

### 3.4 多轮执行不投重复素材
按钮组，二选一：
| 按钮文案 | data-value | 默认 | API 字段 |
|---------|-----------|------|---------|
| 关闭 | `0` | ✓ | `dedup_across_rounds = false` |
| 开启 | `1` | | `dedup_across_rounds = true` |

### 3.5 广告个数
数字输入，不填默认 30。占位提示「不填写默认30」。

### 3.6 素材预览
- 初始态：灰色提示文字「点击「预览素材」查看匹配结果」
- 触发：点击「预览素材」按钮，按钮文案变为「收起预览」
- Loading 态：显示「正在查询匹配素材...」，600ms 模拟延迟
- 结果态：表格展示匹配素材，列含 素材名称、剧本ID、剧本名称、短剧、语言、业务类型、媒体、消耗、DOROI、D0广告ROI、D0 IAA、综合达标率、老达标率、创建时间
- 结果 > 10 条时，仅展示前 10 条，底部提示「共 N 条，仅展示前 10 条」
- 结果 ≤ 10 条时，底部提示「共 N 条」
- 空结果态：显示「未找到匹配的素材」
- 关闭：点击「收起预览」或右上角 ×
- 重置：表单中任何筛选相关字段变化时，自动关闭预览并重置为初始态

## 4. 新建任务 — 投放配置
### 4.1 投放账户
下拉选择已有的账户包，必填。占位文案「请选择账户包」。

### 4.2 广告目标
按钮组，二选一，必填：
| 按钮文案 | data-value | API `objective_type` | 默认 |
|---------|-----------|---------------------|------|
| 应用推广 | `app_promotion` | `APP_PROMOTION` | ✓ |
| 销量 | `product_sales` | `WEB_CONVERSIONS` | |

切换时：
- 应用推广 → 显示 `#appPromotionConfig`，隐藏 `#salesConfig`
- 销量 → 显示 `#salesConfig`，隐藏 `#appPromotionConfig`
- 切换目标触发素材预览重置

## 5. 应用推广配置 (`APP_PROMOTION`)
### 5.1 Smart+ 推广系列
固定文案「已开启」，不可操作。对应 API `campaign_type = "SMART_PLUS"`。

### 5.2 推广系列类型
按钮组，三选一，必填。对应 API `app_promotion_type`：
| 按钮文案 | data-value | API 值 |
|---------|-----------|--------|
| 应用安装 | `app_install` | `APP_INSTALL` |
| 应用再营销 | `app_retargeting` | `APP_RETARGETING` |
| TikTok Minis | `tiktok_minis` | `MINIS` |

**推广系列类型联动规则（表格穷举）：**
| 控制项 | 应用安装 | 应用再营销 | TikTok Minis |
|-------|---------|-----------|-------------|
| 选择应用 | 显示 | 显示 | 隐藏 |
| 选择小程序 | 隐藏 | 隐藏 | 显示 |
| 商品数据源 | 隐藏 | 隐藏 | 显示 |
| 优化目标可选值 | 应用内事件、点击数、安装、价值 | 仅 应用内事件 | 仅 价值 |
| 版位-自动版位 | 可选手动选择 | 可选手动选择 | 强制隐藏 |
| 版位-Pangle | 可选 | 可选 | 强制隐藏+取消勾选 |
| 版位-TikTok | 可选 | 可选 | 强制勾选 |

选择应用下拉：`Netshort` / `Nettoon`（对应 API `app_id`）。
选择小程序下拉：`TT_ZeroDrama(7623607732641417217)` / `TT_FishDrama(7644122628625598471)`。每个小程序绑定固定业务类型，选择小程序后自动确定业务类型（如 ZeroDrama→免费、FishDrama→收费），业务类型不可手动修改。

### 5.3 商品数据源
按钮组，二选一（仅 TikTok Minis 类型可见）：
| 按钮文案 | data-value |
|---------|-----------|
| 关闭 | `0` |
| 开启 | `1` |

选「开启」时显示「关联商品投放」行；选「关闭」时隐藏并清空已选商品。

### 5.4 关联商品投放
显示为链接按钮「关联商品投放」，点击打开商品库选择侧边栏（见 7.3）。已选商品以蓝色标签展示，可逐个移除。关闭商品数据源时清空。

### 5.5 单日预算额度
美元金额输入，必填。前缀 `$` + 后缀 `USD`。对应 API `budget_type = "daily"` + `daily_budget`。

### 5.6 广告系列状态
按钮组，二选一。对应 API `operation_status`：
| 按钮文案 | data-value | API 值 |
|---------|-----------|--------|
| 启用 | `1` | `ENABLE` |
| 暂停 | `0` | `DISABLE` |

### 5.7 广告组设置 — 优化目标
按钮组，对应 API `optimization_goal`。可选值由推广系列类型控制（见 5.2 表格）：
| 按钮文案 | data-value |
|---------|-----------|
| 应用内事件 | `app_event` |
| 点击数 | `click` |
| 安装 | `install` |
| 价值 | `value` |

**优化目标联动：**
- 选择「应用内事件」→ 显示应用内事件下拉：购买(`purchase`)、加入购物车(`add_to_cart`)、查看内容(`view_content`)、注册(`registration`)、达到级别(`reach_level`)
- 选择「价值」→ 显示价值事件下拉：广告收入价值(`ad_revenue`)、应用内购价值(`iap_value`)；若推广系列类型=应用安装，额外显示「目标 ROAS」输入框
- 选择「点击数」或「安装」→ 不显示额外字段

### 5.8 广告组设置 — 竞价策略
按钮组，可选值由优化目标 × 推广系列类型共同决定：
| 优化目标 | 推广系列类型 | 最大转化量(`max_conversion`) | 成本上限(`cost_cap`) | 最大投放量(`max_delivery`) |
|---------|------------|---------------------------|---------------------|-------------------------|
| 应用内事件 | 应用安装/应用再营销 | 隐藏 | 显示 | 显示 |
| 点击数 | 应用安装 | 隐藏 | 显示 | 显示 |
| 安装 | 应用安装 | 隐藏 | 显示 | 显示 |
| 价值 | 应用安装 | 隐藏 | 隐藏 | 仅显示 |
| 应用内事件 | 应用再营销 | 隐藏 | 显示 | 显示 |
| 价值 | TikTok Minis | 隐藏 | 隐藏 | 仅显示 |

- 选「最大投放量」时隐藏出价输入框
- 选「成本上限」或「最大转化量」时显示出价输入框（美元，必填）
- 对应 API `bid_type`

### 5.9 广告组设置 — 版位
按钮组，二选一。对应 API `placement`：
| 按钮文案 | data-value |
|---------|-----------|
| 自动版位 | `auto` |
| 指定版位 | `manual` |

选择「指定版位」时显示复选框组：TikTok、Pangle。TikTok Minis 类型强制「指定版位」且仅 TikTok 可选、Pangle 不可见。

### 5.10 广告组设置 — 定向包
显示当前选中定向包名称（未选择时显示「未选择」）+「选择定向包」链接按钮。点击打开定向包选择侧边栏（见 7.2）。
**联动规则**：商品数据源选择「开启」时，广告组级别定向包隐藏（定向改为商品级别，见 7.3.1）。

### 5.11 广告设置
- **行动号召**：固定文案「立即观看」，不可编辑。对应 API `call_to_action`。
- **用户评论** 按钮组：关闭(`0`, 默认) / 开启(`1`)。对应 API `user_comments`。
- **视频下载** 按钮组：关闭(`0`, 默认) / 开启(`1`)。对应 API `video_download`。
- **视频共享** 按钮组：关闭(`0`, 默认) / 开启(`1`)。对应 API `video_share`。

### 6.1 商品数据源
按钮组，始终可见，二选一：
| 按钮文案 | data-value | 默认 |
|---------|-----------|------|
| 关闭 | `0` | ✓ |

同应用推广 5.4，点击打开商品库选择侧边栏。对应 API `catalog_enabled = true`。

### 6.3 单日预算额度
同应用推广 5.5。

### 6.4 广告系列状态
同应用推广 5.6。

### 6.5 广告组设置 — 广告组名称
文本输入，最大 50 字符，带字数统计。

### 6.6 广告组设置 — 广告组状态
按钮组：启用(`1`) / 暂停(`0`, 默认)。

### 6.7 广告组设置 — 优化目标
按钮组，对应 API `optimization_goal`：
| 按钮文案 | data-value |
|---------|-----------|
| 转化 | `conversion` |
| 点击数 | `click` |
| 落地页浏览量 | `landing_page_view` |
| 价值 | `value` |

**优化目标 → 优化事件联动：**
- 转化 / 点击数 → 显示优化事件下拉（默认「付费」）：下载(`download`)、付费(`purchase`)、加入心愿单(`add_to_wishlist`)、加入购物车(`add_to_cart`)、开始结账(`initiate_checkout`)、查看内容(`view_content`)、订阅(`subscribe`)
- 落地页浏览量 / 价值 → 隐藏优化事件下拉；价值时事件固定为 `purchase`

### 6.8 广告组设置 — 竞价策略
按钮组：
| 优化目标 | 成本上限(`cost_cap`) | 最大投放量(`max_delivery`) |
|---------|---------------------|-------------------------|
| 转化 | 显示 | 显示 |
| 点击数 | 显示 | 显示 |
| 落地页浏览量 | 显示 | 显示 |
| 价值 | 隐藏 | 仅显示 |

- 成本上限 → 显示出价输入框（美元）+ 隐藏目标ROAS
- 最大投放量 → 隐藏出价输入框
- 优化目标=价值时 → 强制最大投放量，显示目标ROAS输入框

### 6.9 广告组设置 — 事件统计方式
按钮组：
| 按钮文案 | data-value | 默认 | API 含义 |
|---------|-----------|------|---------|
| 仅一次 | `once` | ✓ | 仅统计首次事件 |
| 每一次 | `every` | | 统计每次事件 |

### 6.10 广告组设置 — 版位
同应用推广 5.9，但无 TikTok Minis 强制逻辑。使用 `#salesPlacement` 和 `#salesPlacementManualItem`。

### 6.11 广告组设置 — 定向包
同应用推广 5.10（含商品数据源联动隐藏规则）。

### 6.12 广告设置
同应用推广 5.11（行动号召固定「立即观看」，评论/下载/共享开关）。

## 7. 侧边栏
### 7.1 新增账户包
从右侧滑入的抽屉（720px 宽）。
**表单字段：**
- 账号包名称：文本，必填，最大 50 字符，带计数
- 推广渠道：文本，必填
- 渠道号：文本，必填
- 投放策略 按钮组：直投(`直投`, 默认) / W2A(`W2A`)
- W2A 模式下额外显示：
  - 行动号召（落地页文案）：文本，最大 26 字符，带计数
  - 像素ID：文本
- **账户范围**：标题显示账户总数，搜索框可过滤账户名
  - 表格列：账户ID、账户名称、操作（「添加」按钮，已添加显示绿色 ✓「已添加」）
  - 分页
  - 底部「已选账户 (N)」标签区，可逐个移除
- 抽屉底部：取消 / 确定 按钮

### 7.2 定向包
从右侧滑入的抽屉。
**列表视图（默认）：**
- 「+ 新建定向包」按钮
- 已有定向包列表卡片：名称 + 描述，点击选中（蓝色边框高亮），已选中显示「已选择」标记
- 抽屉底部：取消 / 确定（确定后回填到主表单的定向包标签）

**新建表单视图（点击「+ 新建定向包」后切换）：**
- 定向包名称：文本，必填，最大 50 字符
- 性别 按钮组：不限(默认) / 男 / 女
- 年龄 按钮组（多选）：18-24 / 25-34 / 35-44 / 45-54 / 55+

## 9. 全局交互规则
### 9.1 按钮组模式
`btn-group` 容器内 `.btn-option` 按钮通过 `bindBtnGroup(selector)` 统一绑定：
- 点击 → 移除同组所有 `.active` → 当前按钮添加 `.active`
- 触发联动函数（如 `resetMaterialPreview`、`toggleProductLibraryConnect`）
- 部分按钮组有额外自定义 handler，在 `bindBtnGroup` 后单独追加

### 9.2 素材预览重置
以下操作触发 `resetMaterialPreview()`（关闭预览 + 重置为初始态）：
- 筛选模式（全部/自定义）切换
- 业务类型、筛选媒体、创建时间、统计范围、素材数量、排序指标 任一变化
- 筛选条件行增删改
- 广告目标（应用推广/销量）切换
- 推广系列类型切换
- 条件重复校验不通过时

### 9.3 抽屉与模态框层级
| 组件 | z-index | 说明 |
|------|---------|------|
| 模态框 `.modal-overlay` | 1000 | 新建任务表单，居中 |
| 抽屉 `.drawer-overlay` | 1001 | 右侧滑入，覆盖模态框 |
| 定向包抽屉 `#targetingDrawer` | 1002 | 覆盖商品库抽屉，确保从商品库内打开定向选择时可见 |
| 多选下拉 `.multi-select-dropdown` | 10 | 相对定位 |

### 9.4 表单校验
提交时校验项：
- 任务名称 必填
- 执行频率 必填
- 至少选择一个短剧
- 结束运行条件两个数值必填
- 计划创建上限 1-30
- 投放账户 必填
- 素材筛选 = 自定义时至少一个筛选条件

## 10. 生成监测链接
